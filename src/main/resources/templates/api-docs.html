Flowable Spring Demo - API文档

项目说明

这是一个Flowable和Spring Boot集成的入门示例，实现了请假审批流程。

  • 主要功能：请假申请、流程审批、任务管理、流程历史追踪
  • 技术栈：Spring Boot 3.2.5, Flowable 7.0.2, H2 Database
  • 数据库控制台：http://localhost:8080/h2-console
  • API地址：http://localhost:8080/api/leave

请假申请管理

1. 创建请假申请 POST
URL: POST /api/leave
Content-Type: application/json
Body示例:
{ "applicant": "张三", "department": "技术部", "leaveType": "年假", "startDate": "2024-06-25", "endDate": "2024-06-28", "reason": "家庭旅游" }
2. 启动请假流程 POST
URL: POST /api/leave/{id}/start-process
说明: 启动指定ID的请假申请的审批流程
3. 获取所有请假申请 GET
URL: GET /api/leave
4. 获取待办任务 GET
URL: GET /api/leave/tasks
参数: assignee=department_manager (默认)
5. 完成任务 POST
URL: POST /api/leave/tasks/{taskId}/complete
Body示例:
{ "approvalDecision": "approve", "approvalComments": "同意请假" }
6. 获取流程历史 GET
URL: GET /api/leave/process/{processInstanceId}/history

请假审批流程说明

请假审批流程包含以下步骤：

  1. 开始事件：提交请假申请
  2. 部门经理审批：部门经理审批请假申请
  3. 审批决定：根据审批结果决定下一步
  4. 人事备案：审批通过后，人事部门进行备案
  5. 结束事件：流程结束
流程定义文件位置
src/main/resources/processes/leave-request.bpmn20.xml

使用说明

启动应用
mvn spring-boot:run
测试步骤
  1. 启动应用后，会自动创建3个演示请假申请
  2. 使用API创建新的请假申请
  3. 启动请假申请对应的审批流程
  4. 查看待办任务列表
  5. 完成部门经理审批任务
  6. 查看流程历史信息
  7. 在H2控制台查看数据
访问H2数据库
  • URL: http://localhost:8080/h2-console
  • Driver Class: org.h2.Driver
  • JDBC URL: jdbc:h2:mem:testdb
  • User Name: sa
  • Password: (空)

Flowable集成说明

项目主要集成了以下Flowable组件：

  • RuntimeService: 流程运行时服务，负责启动和管理流程实例
  • TaskService: 任务服务，管理用户任务
  • HistoryService: 历史服务，查询流程历史信息
  • RepositoryService: 仓库服务，管理流程定义（隐式使用）
BPMN流程元素
<startEvent> - 开始事件
<userTask> - 用户任务
<exclusiveGateway> - 排他网关
<serviceTask> - 服务任务
<endEvent> - 结束事件
<sequenceFlow> - 顺序流
